## 🚀 How It Works

1. User enters a normal message
2. App processes the text
3. Message converts into encrypted symbol code
4. Encoded message is displayed instantly

Example:

```text
walcome to cyber-encoder-app
↓
~@7$08&_=0_$>#&?-&9$0%&?-@**
```
